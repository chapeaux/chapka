# Commit message footer -- templates

Place these **after** the commit subject and body. Adjust to match **upstream** requirements.

Use `/project:commit-assisted` in Claude Code to draft a commit message with the correct trailer automatically.

## Assisted-by (common)

```
Assisted-by: Claude Code
```

```
Assisted-by: Claude Opus 4.6
```

With model detail when required:

```
Assisted-by: Claude:claude-opus-4-6
```

Some projects distinguish mostly automated output; use their wording, for example:

```
Generated by: <Tool>
```

## DCO sign-off (human only)

Only the **contributor** adds this -- never an automated agent on the contributor's behalf:

```
Signed-off-by: Your Name <your.email@example.com>
```

## Remember

- Do **not** use `Co-authored-by:` for models, products, or bots.
- Match the target project's spelling: `Assisted-by:` vs `Assisted By:` if they specify one.
